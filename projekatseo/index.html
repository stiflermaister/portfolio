To-Do Mini lista zadataka

Ova To-Do Mini lista zadataka je praktičan primer web aplikacije koja koristi HTML, CSS i JavaScript. Omogućava dodavanje zadataka, označavanje kao završeno i brisanje stavki.

    Uputstvo za To-Do listu

    • Unesi tekst i klikni Dodaj (ili Enter).
    • Klik na tekst zadatka označava da je zadatak završen.
    • Dugme ✖ briše zadatak.
    • Zadaci se čuvaju u browseru pomoću localStorage mehanizma.
